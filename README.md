# wiki-gen

> Extendable wiki generator

## Installation

#### Quick

- You'll need `npm` installed on your system

```npm
npm install wiki-gen -g
```

#### From Source / Language tools

- you'll need nodejs installed
```sh
git clone https://github.com/fxnoob/wiki-gen
cd wiki-gen 
npm install -g
```
